Configure iDFit devices in LAN

This project arises from the need to make configurations to biometric devices from the ControlID central, without the need to physically move to their location for different reasons such as the location inside or outside the city.

Default username: admin

CHANGE LANGUAGE

From this section you can modify the language of the device interface and the GUI.

RESET DEVICE

This functionality is very useful to remotely reset devices without losing IP configuration.

RESTART DEVICE

In case the configurations that require it have not done it automatically.

AMAZING FEATURES

Currently I have only been able to apply it to iDFit devices

Customizable

You can modify the code to suit your needs.

Save Your Budget

You will avoid expenses on tickets and transportation.

Remote Management

Without losing IP address to continue using the device.

Save Your Time

No need to physically access the device.
